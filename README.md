# STL Shell Optimizer

A python-based program to create geodesic structures through an optimization algorithm.

Uses Trimesh and Numpy to manipulate mesh structure before recreating into a full mesh for FEniCS through Dolfinx.

Simulates bone flexibility to prevent stress shielding and create custom bone plates using compliance.
